## [0.1.0] - 2025-01-01

### Added
- Multi-provider transcription: AssemblyAI and Deepgram side-by-side
- Pre-recorded transcription via URL or file upload
- Live streaming via WebSocket (AssemblyAI v3, Deepgram)
- Browser microphone support via WebRTC
- Audio quality analysis: waveform, spectrogram, silence/clipping detection
- Provider comparison tab
- Debug / API Inspector tab with raw JSON and cURL export
- Docker support for Hugging Face Spaces deployment
- GitHub Actions CI (pytest) and auto-deploy to HF Spaces
